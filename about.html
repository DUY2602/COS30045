About This Project

A small demonstration website for COS30045 - Data Visualisation Exercise 0.2

Purpose

To help Australian consumers understand and compare energy consumption of household appliances, with a special focus on televisions.

Data Sources

  • Energy Rating Label program – energyrating.gov.au
  • Televisions dataset (monthly CSV releases) – data.gov.au
  • Typical values based on 10 hours/day usage (MEPS standard)

Technologies Used

  • HTML5 for semantic structure
  • CSS3 for responsive styling and layout
  • Vanilla JavaScript for interactivity
  • Responsive Grid & Flexbox layouts
  • CSS Custom Properties (Variables)
